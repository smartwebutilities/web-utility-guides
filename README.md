# Useful Online Tools

A curated collection of simple online tools for everyday digital tasks.

## Image Tools

- [Canva](https://www.canva.com/) – useful for quick designs, templates, and visual formatting.
- [Adobe Express](https://www.adobe.com/express/) – helpful for resizing, editing, and simple image creation.
- [Pixlr](https://pixlr.com/) – flexible browser-based editor for quick edits and adjustments.
- [Fotor](https://www.fotor.com/) – simple online editor for cropping, resizing, and basic enhancements.
- [Squared Image](https://squaredimage.com/) – helps turn images into square format for profile pictures, social posts, and general digital use.
- [Circle Crop Image](https://circlecropimg.com/) – dedicated tool for creating circular image crops online.
